# {{ page.title }}

Diese Seite veröffentlicht Entscheidungen. Keine Analysen, keine Erklärungen, keine Strategiepapiere. Jede Veröffentlichung besteht aus einer kurzen Einordnung und einer begrenzten Liste klar formulierter Entscheidungssätze.

Die Entscheidungen entstehen in Zusammenarbeit mit Personen, die heute politische, militärische oder fachliche Verantwortung tragen. In Gesprächen werden Positionen so verdichtet, dass sie als eindeutige Entscheidungen formuliert werden können, mit klarer Abgrenzung gegenüber ausgeschlossenen Alternativen. Veröffentlicht wird nur, was in dieser Form Bestand hat.

Die veröffentlichten Entscheidungssätze benennen die Entscheidungen, die getroffen werden müssten, damit das jeweils genannte Ziel real werden kann. Sie sind keine formalen Beschlüsse und sagen nichts darüber aus, ob sie bereits institutionell getroffen oder umgesetzt wurden. Wer liest, kann zustimmen, widersprechen oder weitergeben, aber nicht so tun, als seien die zugrunde liegenden Entscheidungen unklar geblieben.
